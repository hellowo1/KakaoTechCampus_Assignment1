# 📝 생산성 Todo 앱
순수 HTML / CSS / JavaScript로 만든 미니멀 할 일 관리 앱입니다.
별도 설치 없이 브라우저에서 바로 실행되며, 데이터는 로컬스토리지에 저장됩니다.

## ✨ 주요 기능

### 기본 CRUD
- 할 일 **추가** (입력 후 버튼 클릭 또는 Enter)
- 할 일 **수정 / 완료 / 삭제**
- 빈 입력 시 안내 메시지 표시
- 완료된 항목은 텍스트에 취소선 표시
- **전체 / 진행 중 / 완료** 탭으로 상태별 필터링
- 현재 선택된 탭 시각적 강조

### 일간 / 주간 뷰
- 상단에 선택된 날짜 표시, 이전 / 다음 버튼으로 날짜 이동
- 주간 달력에서 날짜별 할 일 개수 확인 및 날짜 선택
- 할 일은 **선택된 날짜에 귀속**되어 저장
- 해당 날짜의 할 일만 목록에 표시

### 데이터 저장
- 추가 / 수정 / 삭제 / 완료 시 자동으로 로컬스토리지에 저장
- 새로고침해도 데이터 유지 (자동 복원)
- `JSON.stringify` / `JSON.parse` 사용

## 🚀 실행 방법

### 방법 1: 파일 직접 열기
`index.html` 파일을 브라우저로 열면 됩니다.

### 방법 2: VS Code Live Server (권장)
1. VS Code에서 `todo-app` 폴더 열기
2. 확장 프로그램에서 **Live Server** 설치
3. `index.html` 우클릭 → **Open with Live Server**
4. 브라우저에서 자동으로 열림 (코드 수정 시 자동 새로고침)




## 📁 파일 구조

```
todo-app/
├── index.html   # 화면 구조
├── style.css    # 스타일
└── app.js       # 기능 로직
```
